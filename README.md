# Ist MarcellDavisOS die Zukunft der TUI-basierten Betriebssysteme?
Auf gar keinen Fall. Dies ist nichtmal ein richtiges Betriebssystem, nur ein Programm, welches vorgibt, ein Betriebssystem zu sein. Für ein richtiges Betriebssystem, siehe "[MagentaOS Desktop](https://github.com/MagentaCoding/MagentaOS)" von [MagentaCoding](https://github.com/MagentaCoding).
